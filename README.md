<h2 align="center"> Sistem Informasi Inventory Menggunakan Framework Laravel 5.8</h2>

<h4>Cara Install : </h4>
```bash
git clone https://github.com/ivalrivall/inventory.git
composer install
php artisan migrate
php artisan key:generate
php artisan serve
```
